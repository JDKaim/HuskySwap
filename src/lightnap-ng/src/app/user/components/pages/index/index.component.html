How it Works:

The All Classes page serves as a catalog of all classes available in the system. Each class includes details such as its title, description, schedule, and availability. From this page, users can:

  • Add to Current Classes: Enroll in a class to include it in their active course list.
  • Add to Wishlist: Save a class to their wishlist for future consideration or potential swaps, without committing to enrollment.

The My Classes page is your personal dashboard for managing the courses you are currently enrolled in. This page keeps track of your active courses and offers the following options:

  • Remove a Class: If you no longer wish to remain enrolled in a course, you can drop it from your current classes.
  • Offer a Class for Trade: Select a class from your enrolled list and propose a trade for a class listed on your wishlist.

The My Wishlist page is a list of courses you’re interested in but are not currently enrolled in. From here you can:

  • View Classes: These are classes you've saved from All Classes
  • These classes become options for you to propose swaps via the My Classes page.

The Incoming Swaps page lists all active trade proposals made by other users. Here, you can view details of proposed swaps, including the class being offered and the class requested in return. From here you can:

  • Accept Swap: Agree to the swap, finalizing the exchange of courses.
  • Deny Swap: Decline the offer if the swap doesn't align with your preferences or schedule.

The Outgoing Swaps page tracks all trade proposals you’ve created. For each swap you can:

  • View Swap Status:View the current state of the swap (e.g., pending, accepted, denied).
  • Cancel Swap:If you no longer want to swap, you can cancel any outgoing swaps you've created

Here you can see all the successful swaps you've done.
